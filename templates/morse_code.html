Morse Code: History and Skill

--

Morse code, invented in the 1830s by Samuel Morse and Alfred Vail, is a way to send messages using light and sound.

---

It uses short and long signals called dots . and dashes _.

.-.

Each letter, number, and punctuation mark has its own combination of dots and dashes.

...

Before phones, Morse code was the main way to communicate over long distances.

.

Learning Morse code can be a fun challenge and a way to improve your communication skills.
